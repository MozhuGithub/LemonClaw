# src

该目录存放 LemonClaw 的应用源码。

## 目录结构
- `main/`：Electron 主进程与集成层
- `preload/`：渲染进程与主进程之间的安全桥接层
- `renderer/`：React 前端界面
- `shared/`：主进程与渲染进程共享的类型、常量、IPC 定义和校验结构

## 边界说明

### `main/`
负责：
- 应用生命周期
- 窗口创建
- Gateway 生命周期管理
- 本地产品层服务
- IPC 处理
- 本地配置与存储访问

### `preload/`
负责：
- 向渲染进程暴露安全可控的 API
- 建立主进程与前端之间的桥接

不要把 UI 逻辑放在这里。

### `renderer/`
负责：
- 工作台界面
- 页面与组件
- 前端状态管理
- 用户交互

不要在这里直接访问不安全的 Node / Electron 能力。

### `shared/`
负责：
- 共享类型
- 常量
- IPC 通道定义
- 校验 schema

不要在这里放过重的运行时逻辑。
