# references

该目录用于存放 LemonClaw 的外部参考资料。

## 说明
这里的内容仅用于：
- 阅读
- 对照
- 调研
- 架构理解

这里的代码和资料**不是** LemonClaw 自己的业务源码。

## 当前参考来源
- `openclaw/`
- `homiclaw/`
- `rivonclaw/`
- `hermes/`
- `clawx/`

## 使用原则
- 不要把这里的代码当成 LemonClaw 的实现代码直接修改
- 不要把参考代码和 LemonClaw 的正式源码混放
- 如果需要借鉴某个思路，应在 LemonClaw 自己的 `src/` 中重新实现
